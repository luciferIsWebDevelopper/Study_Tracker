# Study Tracker for MLOps 🚀

## 📌 Problem

While learning MLOps and programming, maintaining consistency and breaking down large topics into daily actionable tasks is challenging.

## 💡 Solution

This application converts structured learning content into tasks automatically and helps users stay consistent by tracking their daily progress and streak.

---

## ✨ Key Features

* 📥 Paste structured content → auto-generate tasks
* 🗂️ Folder creation using custom rules (`~~~`)
* ✅ Task extraction using `[ ]` pattern
* 🔥 Daily streak tracking
* 🎯 Focused and structured learning workflow

---

## 🧠 How It Works

* Users paste structured text input
* The system parses:

  * `~~~` → creates folders/topics
  * `[ ]` → creates tasks
* Tasks are dynamically generated and displayed
* Users complete tasks daily to maintain streak

---

## 🚀 Example Input

```
Programming
[ ] Learn Python basics
[ ] Practice loops
```
